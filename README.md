# LeadFlow AI

Landing page pública da Fase 1 do LeadFlow AI, um SaaS focado em vendedores que vivem no WhatsApp.

## Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- shadcn/ui

## Fase 1

Marketing site mobile-first com:

- hero section
- problema
- solução
- recursos
- prévia de preços
- FAQ
- CTA final
